## 三、本作品对常见质疑的诚实回应

国奖评审常见三大质疑，我们正面回应：

### 质疑 1："你们的 LightGBM MAE = 0.56，是不是数据泄漏？"

**承认**：是的。这是因为 v1 实验中使用了 `pm25_lag_1h`（上一小时真实值）作为特征。  
**解决**：v2 已剔除强 lag 特征，所有模型在公平实验设置下重新对比，详见报告 5.2 节、表 5-1。

### 质疑 2："MSTN 的'空间卷积'在单城市数据上有什么意义？"

**承认**：v1 的命名误导。"空间卷积"实际是沿特征通道的 1D 卷积。  
**解决**：v2 将该模块**重命名为 `FeatureCorrelationAttention`（特征相关注意力）**，命名诚实。

### 质疑 3："你们的多城市数据是真实的吗？"

**承认**：UCI 数据集只有北京单站点。  
**解决**：v2 提供 [`real_multi_city_collector.py`](code/real_multi_city_collector.py)：
- 优先尝试 OpenAQ v3 真实多城市数据
- 失败回退到"北京 5 站点合成扩展"，并在 source 列**明确标注 `UCI_extended_synthetic`**
- 报告 4.2 节如实说明数据来源

## 四、核心创新点速览

### 创新 1：MSTN v2 多尺度时空融合网络

| 维度 | 现有方法 | MSTN v2 |
|---|---|---|
| 时间建模 | 单尺度 LSTM | 三尺度因果膨胀 TCN（dilation=1/4/8） |
| 尺度融合 | 静态加权 | **跨尺度自注意力**（动态自适应） |
| 空间建模 | 依赖邻接矩阵的 GCN | **特征相关注意力**（自学习，无需邻接矩阵） |
| 输出形式 | 单点估计 | **分位数预测 q05/q50/q95**（90% 置信区间） |
| 参数量 | ST-GCN 100万+ | 80K（远低于 100 万限制） |

### 创新 2：BHI 呼吸健康指数

```
BHI = 0.55 × IPM + 0.15 × IT + 0.30 × IE
       │           │           │
       │           │           └─ 基于 WHO 2021 全球空气质量指南
       │           └─ 基于 ASHRAE 55-2020 热舒适标准
       └─ 基于 GB 3095-2012 IAQI 分段非线性映射
```

权重 0.55/0.15/0.30 来自文献相关性分析（详见报告附录 A，引用 5 篇 SCI 论文）。

### 创新 3：全流程一键 Pipeline + 工业级 Web 应用

- **12 个 Python 模块，~6,500 行核心代码**
- `python run.py` 一条命令完成 10 步从数据采集到论文图表
- Streamlit Web 应用 1,439 行，深色科技风，6 大功能面板
- 一份 [`smoke_test.py`](smoke_test.py) 10 秒验证所有模块可用

## 五、关键实验结果速览

> 完整数据见报告第 5 章。下表为 1h 提前量、公平实验设置（剔除强 lag 特征）。
