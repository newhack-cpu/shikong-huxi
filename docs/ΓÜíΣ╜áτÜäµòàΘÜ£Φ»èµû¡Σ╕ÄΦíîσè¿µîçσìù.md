## 五、环境问题的常见困惑解答

### Q1: 为什么不能直接用我的 anaconda31 base 环境？
A: 你的 base 环境是 Python 3.7（2018 年）+ sklearn 0.19（2017 年）。这些库**在 2026 年的 PyTorch/MSTN/Streamlit 里很多 API 都已经被移除**。强制用会出现各种"莫名其妙"的报错。**用独立环境是工程最佳实践**。

### Q2: 创建 conda 环境会不会影响我现有的项目？
A: 完全不会。conda 环境之间是隔离的。你的 base 环境一切照旧，只是多一个叫 `shikong` 的新环境。

### Q3: 我磁盘够吗？
A: 新环境约 4-5GB（PyTorch 占大头）。你 D 盘应该够。

### Q4: 没有网络怎么办？
A: 用 `offline_data_generator.py` 生成合成数据。所有后续 pipeline 都能正常跑。**注意正式提交前要换真实数据**。

### Q5: 跑出来 MAE 数字是不是和报告里的预期值一样？
A: **不一定一样**。报告里 MAE=10.84 是 v1 实验的预期值。你跑出真实结果后，**用真实数字替换报告/PPT/答辩稿里的所有占位**。
