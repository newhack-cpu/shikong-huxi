## 🆘 如果遇到问题怎么办

### Q: 真消融实验跑不动（OOM 或太慢）

A: 用 `--quick` 模式（10 epoch、1 次重复），或者降低 batch_size：
```python
# 在 8_ablation_study_v2.py 中
trainer = train_one_variant(..., batch_size=32)
```

### Q: OpenAQ 拉不到数据

A: 三种回退：
1. 申请 OpenAQ 免费 API key（注册即得，无审核）
2. 使用国家空气质量在线监测发布平台（需要 https + 客户端证书，复杂）
3. 接受 honest_fallback 用"北京 5 站点扩展"，**报告中如实说明**

### Q: PPT 在评委电脑上字体不显示

A: 三个保险措施：
1. 把 .pptx 嵌入字体（PowerPoint → 选项 → 保存 → 嵌入字体）
2. 同时准备一份 PDF 版本
3. 自带笔记本演示

### Q: Web 应用现场崩溃

A: 录一个 30 秒的"Web 应用功能演示视频"作为 Plan B；
   现场如果跑不起来，直接放视频。
